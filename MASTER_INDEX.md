## 🚀 الأوامر الأساسية

### البدء
```bash
npx prisma migrate dev --name init_schema
npx tsx prisma/seed.ts
npx prisma studio
```

### التطوير
```bash
npm test database
npm run dev
git commit -m "add feature"
```

### التحقق
```bash
npx prisma validate
npx prisma format
npx prisma generate
```

### الإنتاج
```bash
npx prisma migrate deploy
npm run build
npm run start
```
